## Quick Start

### Option 1: Script Tag

Add this single line to your HTML:

```html
<script src="https://unpkg.com/@supprt/widget" data-public-key="pk_xxx"></script>
```

### Option 2: npm Package

```bash
npm install @supprt/widget
```

```javascript
import { init } from '@supprt/widget'

init({
  publicKey: 'pk_xxx'
})
```
